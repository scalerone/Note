# \~否(非)运算

## 符号

*   `~`

## 写法

*   `~整数`

*   将该整数按位取反

```javascript
//  1    0000 0000 0000 0000 0000 0000 0000 0001
// 结果  1111 1111 1111 1111 1111 1111 1111 1110
~1 // -2
```

## 实际应用

*   快速取整

    ```javascript
    ~~1.2 // 1

    ~~20.3 // 20
    ```

*   取反：数字 - 1

    ```javascript
    ~2 // -3

    ~-2 // 1

    ~0 // -1

    ~NaN // -1
    ```
