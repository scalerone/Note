# 多重判断

## 使用 Array.includes

*   常规

    ```javascript
    function test(fruit) {
      if (fruit == 'apple' || fruit == 'strawberry') {
        console.log('red');
      }
    }
    ```

*   使用 `Array.includes`

    ```javascript
    function test(fruit) {
      const redFruits = ['apple', 'strawberry', 'cherry', 'cranberries'];

      if (redFruits.includes(fruit)) {
        console.log('red');
      }
    }
    ```
