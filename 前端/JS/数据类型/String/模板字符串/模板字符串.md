# 模板字符串

## 模板字符串

*   添加变量 `${变量}`

    ```javascript
    var xm = {
      age:18,
      height:180,
      name:xiaoming,
    }
    var {age,name,height} = xm;
    var st = `name:${name} age:${age} height:${height}`;
    ```

*   添加表达式

    ```javascript
    var x = 3,
      y=5;
    var st = `${x} + ${y} = ${x+y}`; //打印 "3 + 5 = 8"

    function getNa() {
      return "abc";
    }
    var str = `my ${getNa()}`; //打印 "my 'abc'"
    ```

*   内部使用  `${}` ,可以使用转义符 `\`
