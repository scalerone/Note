# 基础

## 函数声明

  - 函数声明被提升到了函数定义

  - 你可以在函数声明之前使用该函数:

    ```javascript
    hoisted(); // "foo"

    function hoisted() {
      console.log("foo");
    }
    ```

    ```javascript
    var hoisted;

    hoisted = function() {
      console.log("foo");
    }
    hoisted();
    // "foo"
    ```

## 函数表达式

  - 函数表达式不会被提升

  - 函数是一个引用类型，将其赋值给某个变量时，变量中报错的时函数的地址

    ```javascript
    notHoisted(); // TypeError: notHoisted is not a function

    var notHoisted = function() {
      console.log("bar");
    };
    ```
