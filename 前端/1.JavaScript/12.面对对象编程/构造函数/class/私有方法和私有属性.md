# 私有方法和私有属性

## 私有方法和私有属性

  - 私有方法是常见需求，但 ES6 不提供，只能通过变通方法模拟实现

    1.  一种做法是在命名上加以区别

    ```javascript
    class Widget {
      // 公有方法
      foo (baz) {
        this._bar(baz);
      }

      // 私有方法
      _bar(baz) {
        return this.snaf = baz;
      }

      // ...
    }
    ```

    1.  索性将私有方法移出模块，因为模块内部的所有方法都是对外可见的

    ```javascript
    class Widget {
      foo (baz) {
        bar.call(this, baz);
      }
      // ...
    }

    function bar(baz) {
      return this.snaf = baz;
    }
    ```
