# if...else

## 结构

+ `if` 代码块后面，还可以跟一个 `else` 代码块，表示不满足条件时，所要执行的代码

    ```js
    if (m === 3) {
      // then
    } else {
      // else
    }
    ```

+ 多个 `if...else` 语句可以连写在一起

    ```js
    if (m === 0) {
      // ...
    } else if (m === 1) {
      // ...
    } else if (m === 2) {
      // ...
    } else {
      // ...
    }
    ```
