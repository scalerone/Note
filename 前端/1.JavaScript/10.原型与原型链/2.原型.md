# 原型

## 原型 prototype

+ 所有函数都有一个属性：`prototype` 原型

+ 默认情况下，`prototype` 是一个普通的 `Object` 对象

+ 默认情况下 `prototype` 有一个属性 `constructor`，它也是一个对象，它指向构造函数本身

  ```js
  // 默认情况下
  prototype = {
    constructor: {

    }
  }
  ```

  ```js
  function test() {}

  const obj = new test()

  test.prototype.constructor === test // true
  ```
