# GO 、VO 、AO

## 全局VO

+ 也可称之为 `GO`

  + GO：（`Global Object`）全局执行上下文中的VO

+ 声明的变量
+ 声明的函数

## 函数VO

1. 确定所有的形参以及特殊变量 `arguments`
2. 确定函数中通过 `var` 声明的变量，将它们的值设置为 `undefined`。如果VO中已有该名称，则直接忽略
3. 确定函数中通过字面量声明的函数，将它们的值设置为指向函数对象，如果VO中已存在该名称，则覆盖

  ```js
  // 字面量
  function b() {}

  // 函数表达式
  var c = function() {}
  ```

## AO

+ AO(`Active Object`) 当前正在执行的上下文中的VO
