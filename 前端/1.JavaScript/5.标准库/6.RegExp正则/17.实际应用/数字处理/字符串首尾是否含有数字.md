# 字符串首尾是否含有数字

## 代码

+ 检验一个字符串首尾是否含有数字

  ```js
  const str = '123abc';
  const reg = /^\d|\d$/g;

  console.log(reg.test(str)); // true
  console.log(str.match(reg)); // [ '1' ]

  const str1 = 'a123abc5';

  console.log(reg.test(str1)); // true
  console.log(str1.match(reg)); // [ '5' ]

  const str2 = 'a123abc';

  console.log(reg.test(str2)); // false
  console.log(str2.match(reg)); // null
  ```
