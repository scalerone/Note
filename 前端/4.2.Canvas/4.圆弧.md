# 圆弧

## 概述

+ 圆心(x,y)
+ 半径(r)
+ 弧度(起始弧度,结束弧度)
+ 方向(顺时针、逆时针)

## arc()

+ 语法 `ctx.arc(x,y,r,sAngle,eAngle,counterclockwise);`

  + x,y：圆心坐标
  + r：半径大小
  + Angle:绘制开始的角度 (圆心到最右边点是 0 度，顺时针方向弧度增大)
  + eAngel:结束的角度（注意是弧度）
  + counterclockwise：是否是逆时针。true 是逆时针，false：顺时针（默认）

+ 弧度 = (Math.PI/180) * 角度

  ```js
  ctx.arc(100,100, 50,0,Math.PI*1.8);
  ctx.stroke(); // 画线
  ```
