# bigInt

## 概述

  - ES2020 引入原始类型 `BigInt`，用于表示非常大的整数

    ```typescript
    const oneHundred: bigint = BigInt(100);

    // Creating a BigInt via the literal syntax
    const anotherHundred: bigint = 100n;
    ```

## 注意

  - `number` 和 `bigint` 都表示数字，但是这两个类型不兼容

    ```javascript
    let big: bigint =  100n;
    let num: number = 6;
    big = num;
    num = big;

    // 会抛出一个类型不兼容的 ts(2322) 错误。
    ```
