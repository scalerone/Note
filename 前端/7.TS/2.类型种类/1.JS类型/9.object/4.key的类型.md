# key 的类型可以不是 String 吗

## 答案

  + `string`

    ```js
    type A = {
      [k: string]: number
    }
    const a:A = {
      a: 1
    }
    ```

  + `number`

    ```js
    type A = {
      [k: number]: number
    }
    const a:A = {
      1: 1
    }
    ```

  + `symbol`

    ```js
    type A = {
      [k: symbol]: number
    }
    const s = Symbol()
    const a:A = {
      [s]: 1
    }
    ```
