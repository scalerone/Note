# 定义类型

## 基础类型

  - 定义

    ```typescript
    type Name = string;

    ```

  - 对象

    ```typescript
    type PartialPointX = {x:number;};
    type PartialPointY = {y:number;};

    type Point = {
      x:number;
      y:number;
    }

    ```

## 元组

  - 定义

    ```typescript
    type Data = [number,string,boolean];
    ```

## 函数

  - 定义

    ```typescript
    type Callback = (data:string) => void;
    ```

## 联合类型

  - 代码

    ```typescript
    type StringOrNumber = string | number;

    ```

    ```typescript
    type PartialPointX = {x:number;};
    type PartialPointY = {y:number;};

    type PartialPoint = PartialPointX | PartialPointY;
    ```
