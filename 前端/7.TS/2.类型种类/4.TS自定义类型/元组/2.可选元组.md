# 可选元组

## 可选

  - 元组也支持了在某一个位置上的可选成员

    ```ts
    const arr6: [string, number?, boolean?] = ['linbudu'];

    // 下面这么写也可以
    // const arr6: [string, number?, boolean?] = ['linbudu', , ,];
    ```

  - 长度属性也会发生变化

    ```ts
    const arr6: [string, number?, boolean?] = ['linbudu'];

    // 元组的长度可能为 1、2、3
    type TupleLength = typeof arr6.length; // 1 | 2 | 3
    ```
