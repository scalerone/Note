# 专用工具类型

## 概述

  - Uppercase 字符串大写
  - Lowercase 字符串小写
  - Capitalize 首字母大写
  - Uncapitalize 首字母小写

    ```ts
    type Heavy<T extends string> = `${Uppercase<T>}`;
    type Respect<T extends string> = `${Capitalize<T>}`;

    type HeavyName = Heavy<'linbudu'>; // "LINBUDU"
    type RespectName = Respect<'linbudu'>; // "Linbudu"
    ```
