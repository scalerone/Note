# 按比例设置

## 等比例

+ 代码

  ```css
  article {
    width: 300px;
    height: 300px;
    border: 1px solid silver;
    display: grid;

    /* 行高 分成3行 每个 平均（100px） 高 */
    grid-template-rows: repeat(3, 1fr);
    grid-template-columns: repeat(3, 1fr);
  }
  ```

## 按照比例划分

+ 代码

  ```css
  article {
    width: 300px;
    height: 300px;
    border: 1px solid silver;
    display: grid;

    /* 行高 分成3行 比例是 1:2:1 */
    grid-template-rows: 1fr 2fr 1fr;
    grid-template-columns: 1fr 2fr 1fr;
  }
  ```
