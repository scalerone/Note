# 申明变量

## 语法

+ `@type {typeName}`

+ @type 标记允许您提供一个类型表达式，用于标识符号可能包含的值类型或函数返回的值类型
+ 还可以将类型表达式包含在许多其他 JSDoc 标记中，比如 `@param` 标记

+ 申明

  ```js
  /**
   * 使用 "@type" 来声明类型
   * @type {string}
   */
  let var1;
  ```

  ```js
  /** @type {Window} */
  let var2;
  ```

  ```js
  /** @type {number} */
  /**@type {string} */
  /** @type {boolean}*/
  /** @type {RegExp} */
  ```

## 多类型

+ 多类型

  ```js
  /**
   * 可以指定 union 类型，如字符串或者布尔值
   * @type {(string | boolean)}
   */
  let var3;
  ```
