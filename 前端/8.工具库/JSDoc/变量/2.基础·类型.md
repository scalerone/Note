# 基础类型

## 数组

+ 数组

  ```js
  /**
   * 声明元素类型是数字的数组 - 方式1
   * @type {number[]}
   */
  let var4;
  ```

## 对象

+ 对象

  ```js
  /**
   * 声明对象类型
   * @type {{a: string, b: number}}
   */
  let var7;
  ```

## any 类型

+ any 类型

  ```js
  // 方式1
  /** @type {*} - can be 'any' type */
  var star = true;
  ```

  ```js
  // 方式2
  /** @type {?} - unknown type (same as 'any') */
  var question = true;
  ```

  ```js
  // 方式3
  // 都等价于TypeScript类型标注语法
  /** @type {any} */
  var thing = true;
  ```
