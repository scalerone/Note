# 不同方向的投影视图

## x轴方向观察

+ x轴方向观察

  ```js
  // 通过UI按钮改变相机观察角度
  document.getElementById('x').addEventListener('click', function () {
    camera.position.set(500, 0, 0); //x轴方向观察
    camera.lookAt(0, 0, 0); //重新计算相机视线方向
  })
  ```
