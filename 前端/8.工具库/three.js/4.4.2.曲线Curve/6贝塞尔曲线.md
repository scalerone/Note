# 贝塞尔曲线

## 概述

+ 对于一些不规则的曲线，很难用一个圆、椭圆或抛物线函数去描述，这时候，可以使用threejs提供的*样条曲线*或贝*塞尔曲线*去表达

  ![样条曲线与贝塞尔曲线](images/样条曲线与贝塞尔曲线.png)

## 分类

+ 二次贝塞尔曲线

  + 起点、终点和1个控制点定义
  + 2D - QuadraticBezierCurve
  + 3D - QuadraticBezierCurve3

+ 三次贝塞尔曲线

  + 起点、终点和2个控制点定义
  + 2D - CubicBezierCurve
  + 3D - CubicBezierCurve3
