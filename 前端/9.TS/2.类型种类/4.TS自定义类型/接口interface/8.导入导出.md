## 导入

+ 仅仅导入 / 导出声明

  ```js
  // 让编译器知道您要导入的内容绝对是一种类型
  import type T from './mod';
  import type { A, B } from './mod';
  import type * as Types from './mod';

  // 仅用作导出
  export type { T };
  export type { T } from './mod';

  ```

  ```js
  export type { Track, Playlist } from "./types";
  ```
