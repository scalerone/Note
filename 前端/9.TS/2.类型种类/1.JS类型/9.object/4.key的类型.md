# key 的类型可以不是 String 吗

## 答案

+ `string`

  ```ts
  type A = {
    [k: string]: number
  }
  const a:A = {
    a: 1
  }
  ```

+ `number`

  ```ts
  type A = {
    [k: number]: number
  }
  const a:A = {
    1: 1
  }
  ```

+ `symbol`

  ```ts
  type A = {
    [k: symbol]: number
  }
  const s = Symbol()
  const a:A = {
    [s]: 1
  }
  ```
