# 类型断言与类型声明

## 概述

+ 类型断言

 ```ts
  interface Car {
    name: string;
    run():void
  }

  function getCacheData(key:string):any {
    return (window as any).cache[key]
  }

  const tom = getCacheData('tom') as Car

   tom.run()
  ```

+ 类型声明

 ```ts
   interface Car {
    name: string;
    run():void
  }

  function getCacheData(key:string):any {
    return (window as any).cache[key]
  }

  const tom:Car = getCacheData('tom')

  tom.run()
  ```
