# 解决默认导入错误

## 解决办法1

*   示例

    ```javascript
    import fs from "fs"; // 报错 提示：没有默认导出

    fs.readFileSync("./xgg.ts")
    ```

## 解决办法2

*   示例

    ```typescript
    import * as fs from "fs";

    fs.readFileSync("./xgg.ts");
    ```

## 解决办法3

*   在 `tsconfig.json` 设置 `esModuleInterop: true`

    ```typescript
    import fs from "fs";
    ```
