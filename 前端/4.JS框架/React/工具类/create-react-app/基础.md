## 官网

*   地址

    ```javascript
    https://create-react-app.dev/
    ```

## 安装

*   npx

    ```javascript
    npx create-react-app my-app
    ```

*   npm

    ```javascript
    npm init react-app my-app
    ```

*   Yarn

    ```javascript
    yarn create react-app my-app
    ```

## 命令

*   `npm start` or `yarn start`

*   `npm run build` ：打包上线

*   `npm test` ：开启测试环境

*   `npm run eject` ：移除所有工具，**不可回退**
