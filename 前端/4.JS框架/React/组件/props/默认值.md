# 默认值

## 类组件

*   设置

    ```javascript
    class Greeting extends React.Component {
      static defaultProps = {
        age: 18
      }
    }

    ```

## 函数组件

*   设置

    ```javascript
    const Child = ({ type }) => {
      useEffect(() => {
        console.log("type", type);
      }, [type]);

      return <div>Child</div>;
    };

    Child.defaultProps = {
      type: { a: 1 }
    };
    ```
