# 自定义组件 v-mode

## 概述

- 当对自定义组件使用 `v-mode` 指令时，绑定的属性名由原来的 `value` 变成了 `modelValue`，事件名由原来的 `input` 变成了 `update:modelValue`

    ```js
    // vue 2.x
    <cheak-editor
      :value="pageTitle"
      @input="pageTitle = $event"
    />
    // 简写
    <cheak-editor
      v-model="pageTitle"
    />

    // 子组件
    props: {
      value: String
    }
    ```

    ```js
    // vue 3.x
    <cheak-editor
      :modelValue="pageTitle"
      @update:modelValue="pageTitle = $event"
    />
    // 简写
    <cheak-editor
      v-model="pageTitle"
    />

    // 子组件
    props: {
      modelValue: String
    }
    ```

- 去掉了 `.sync` 修饰符，它的功能由 `v-mode` 的参数替代

    ```js
    // vue 2.x
    <cheak-editor
      :title="pageTitle"
      @update.title="pageTitle = $event"
    />
    // 简写
    <cheak-editor
      :title.sync="pageTitle"
    />
    ```

    ```js
    // vue 3.x
    <cheak-editor
      :title="pageTitle"
      @update.title="pageTitle = $event"
    />
    // 简写
    <cheak-editor
      v-model.title="pageTitle"
    />
    ```
