# PropType

## 概述

+ 用于在用运行时 `props` 声明时给一个 `prop` 标注更复杂的类型定义

  ```js
  import { PropType } from 'vue'

  interface Book {
    title: string
    author: string
    year: number
  }

  export default {
    props: {
      book: {
        // 提供一个比 `Object` 更具体的类型
        type: Object as PropType<Book>,
        required: true
      },
      // 也可以标记函数
      callback: Function as PropType<(id: number) => void>
    }
  }
  ```
