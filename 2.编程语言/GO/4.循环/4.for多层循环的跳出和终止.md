# for多层循环的跳出和终止

## 概述

+ 对 `for` 进行打标签，然后通过 `break` 和 `continue` 就可以实现多层循环的跳出和终止

## continue label

+ 跳出当前循环

  ```go
  f1:
    for i := 1; i < 3; i++ {
      for j := 1; j < 5; j++ {
      if j == 3 {
        continue f1
      }
      fmt.Println(i, j)
      }
    }

  >>> 输出
  1 1
  1 2
  2 1
  2 2
  ```

## break label

+ 终止当前循环

  ```go
  f1:
    for i := 1; i < 3; i++ {
      for j := 1; j < 5; j++ {
      if j == 3 {
        break f1
      }
      fmt.Println(i, j)
      }
    }

  >>> 输出
  1 1
  1 2
  ```
