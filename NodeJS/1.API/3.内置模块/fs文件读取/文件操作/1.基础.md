# 基础

## 主要 API

+ `fs.readFile`：异步地读取文件的全部内容

+ `fs.writeFile`：向文件写入内容

+ `appendFile`: 追加的方式向指定的文件中写入数据

+ `copyFile`: 将某个文件中的数据拷贝至另外一个文件

+ `fs.stat`：获取文件或目录信息

+ `fs.readdir`：获取目录中的文件和子目录

+ `fs.mkdir`：创建目录

+ `fs.unlink`：删除文件

+ `fs.access`：判断一个文件是否存在、是否可读、是否可写

+ 文件监控: `fs.watchFile` 比 `fs.watch` 低效，但更好用
