# 空白符

## 分类

  - `\r` 回车符

  - `\n` 换行符

  - `\f` 换页符

  - `\t` 制表符 `tab` 键

  - `\v` 垂直制表符

## \ 转义特殊字符

  - 转义符可以将特殊字符转义

    ```javascript
    // 转义 \
    /\/
    ``
    ```

## 转义

  - 所谓元字符，就是正则中有特殊含义的字符。

  - 所有结构里，用到的元字符总结如下：

    ```javascript
    ^ $ . * + ? | \ / ( ) [ ] { } = ! : - ,
    ```

  - 当匹配上面的字符本身时，可以一律转义

    ```javascript
    const string = "^$.*+?|\\/[]{}=!:-,";
    const regex = /\^\$\.\*\+\?\|\\\/\[\]\{\}\=\!\:\-\,/;
    console.log( regex.test(string) );
    // => true
    ```
