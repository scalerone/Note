# 双向绑定

## 变化

  - 2.x 上线绑定 `v-mode` 和 `.sync` 修饰符

  - 3.x 去掉了 `.sync` 。只保留的 `v-mode`

## 2.x

  - 示例

    ```javascript
    <cheak-editor
      v-model="xx"
      :text.sync="1212"
    />
    ```

## 3.x

  - 示例

    ```javascript
    <cheak-editor
      v-model="xx"
      v-model.text="1212"
    />
    ```
