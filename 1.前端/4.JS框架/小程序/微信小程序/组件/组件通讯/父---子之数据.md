# 父-->子之数据

## 使用

*   父组件

    ```javascript
    <d-bingImg img-src="xxx">
    ```

*   子组件

    ```javascript
    properties: {
      imgSrc: {
        type: String,
        value: 'xxx',
        observer() {}
      }
    }
    ```

    ```javascript
    <view>
      <image src="{{ imgSrc }}" />
    </view>
    ```
