# 基础

## 概述

  - 与联合类型相似
  - 符号 `&`
  - `A & B`需要同时满足 A 与 B 两个类型

## 对象交叉类型

  - 同时符合这两个对象的结构

    ```typescript
    interface NameStruct {
      name: string;
    }

    interface AgeStruct {
      age: number;
    }

    type ProfileStruct = NameStruct & AgeStruct;

    const profile: ProfileStruct = {
      name: "linbudu",
      age: 18
    }
    ```
