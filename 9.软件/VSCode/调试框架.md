## 调试TS

  - 使用 `TS-node` `npm install -D ts-node`

  - 开启 sourceMap

    ```json
    {
      "sourceMap": true,
      "outDir": "./out"
    }
    ```

  - 配置

    ```json
    {
      // 使用 IntelliSense 了解相关属性。
      // 悬停以查看现有属性的描述。
      // 欲了解更多信息，请访问: https://go.microsoft.com/fwlink/?linkid=830387
      "version": "0.2.0",
      "configurations": [
        {
          "type": "node",
          "request": "launch",
          "name": "Launch Program",
          "runtimeArgs": [
            "-r",
            "ts-node/register"
          ],
          "args": [
            "${workspaceFolder}/src/index.ts"
          ]
        }
      ]
    }
    ```
