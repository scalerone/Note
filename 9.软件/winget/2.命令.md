# 命令

## 支持命令

| 命令          | 说明        | 示例                         |
| ----------- | --------- | -------------------------- |
| `search`    | 搜索        | `winget search <appname>`  |
| `install`   | 安装        | `winget install <appname>` |
| `list`      | 显示已安装的包   | `winget list`              |
| `upgrade`   | 升级指定的应用程序 |                            |
| `uninstall` | 卸载给定的包    |                            |
| `show`      | 查看详情      | `winget show skype`        |
|             |           |                            |

![安装](image/安装.png)

## 选项

| 选项               | 描述                                      | 示例          |
| ---------------- | --------------------------------------- | ----------- |
| `-v`、`--version` | 返回 winget 的当前版本                         | `winget -v` |
| `--info`         | 提供有关 winget 的所有详细信息，包括许可证链接、隐私声明和配置的组策略 |             |
| `-?` 、`--help`   |                                         |             |
